# ML-AI Studio User Guide

## Getting Started

### Registration and Login

1. Navigate to the registration page
2. Create an account with your email, username, and password
3. Log in to access the platform

### Dashboard

The dashboard provides an overview of:
- Your projects
- Datasets
- ML models
- Active experiments

## Data Management

### Uploading Datasets

1. Go to **Data Management**
2. Drag and drop your file or click to browse
3. Supported formats: CSV, JSON, Excel, Parquet
4. Maximum file size: 100MB

### Viewing Dataset Summary

1. Click on a dataset
2. View statistical summary including:
   - Number of rows and columns
   - Data types
   - Missing values
   - Basic statistics

### Creating Visualizations

1. Select a dataset
2. Choose chart type (bar, line, scatter, histogram, heatmap)
3. Select columns for X and Y axes
4. Generate interactive charts

## Model Builder

### Creating a Model

1. Go to **Model Builder**
2. Click **Create Model**
3. Fill in:
   - Name
   - Description
   - Model type (classification, regression, clustering, time series)
   - Algorithm (Random Forest, XGBoost, Neural Network, SVM)

### Training a Model

1. Select a model
2. Click **Train Model**
3. Configure:
   - Dataset
   - Target column
   - Test size
   - Hyperparameters
4. Monitor training progress
5. View metrics after training completes

## Templates

Browse pre-built templates for common ML tasks:
- Image Classification
- Text Classification
- Time Series Forecasting
- Customer Churn Prediction
- Price Prediction

Click **Use Template** to get started with a template.

## AI Tools

### LLM Chat

1. Go to **AI Tools** > **LLM Chat**
2. Select a model (GPT-3.5, GPT-4, Claude, etc.)
3. Type your message
4. Get AI-generated responses

### RAG (Retrieval Augmented Generation)

1. Upload documents (PDF, DOCX, TXT, MD)
2. Documents are automatically processed and indexed
3. Query your documents using natural language
4. Get answers based on your document content

## Generative AI Studio

### Image Generation

1. Go to **Generative AI** > **Image Generation**
2. Enter a prompt
3. Select style presets
4. Generate images using Stable Diffusion, DALL-E, or Midjourney

### Audio Generation

1. Text-to-speech with multiple voice options
2. Music generation with style parameters
3. Audio editing tools

### Video Generation

1. Text-to-video generation
2. Image-to-video animation
3. Video editing timeline

## Jupyter Notebooks

1. Go to **Notebooks**
2. Click **Open JupyterLab**
3. Access full JupyterLab environment
4. Create and run Python notebooks
5. Install packages using conda or pip

## Best Practices

1. **Data Quality**: Always clean and preprocess your data before training
2. **Train-Test Split**: Use appropriate test sizes (typically 20-30%)
3. **Hyperparameter Tuning**: Experiment with different hyperparameters
4. **Model Evaluation**: Review multiple metrics, not just accuracy
5. **Version Control**: Keep track of model versions and experiments

## Support

For issues or questions, please refer to the documentation or contact support.
